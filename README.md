# Chocolate Shipments Analysis

## Overview
This project analyzes chocolate shipment data using SQL, Excel, and Power BI. The objective is to identify sales trends, shipment performance, top-selling products, and country-wise revenue.

## Dataset
The dataset contains:
- Shipment ID
- Order Date
- Product
- Country
- Sales Person
- Boxes Shipped
- Amount
- Cost
- Profit

## Tools Used
- SQL
- Microsoft Excel
- Power BI

## Key Insights
- Monthly sales trend
- Top-performing countries
- Best-selling chocolate products
- Profit analysis
- Shipment performance dashboard
